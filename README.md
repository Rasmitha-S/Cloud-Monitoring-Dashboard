About the Project
This project is a cloud monitoring dashboard that displays real-time infrastructure metrics like CPU usage, memory, network I/O, disk storage, and active alerts — similar to tools used in real cloud environments like AWS CloudWatch and Azure Monitor.
Built as part of my cloud engineering internship to demonstrate understanding of cloud infrastructure monitoring concepts.

Features
Live CPU, Memory and Network charts that auto-update every 3 seconds
Resource usage gauges with color-coded thresholds
Running services table with instance IDs and status badges
Alert log with critical, warning and resolved alerts
Region health monitor with live latency ping
Real-time clock
Fully responsive — works on mobile and desktop

Built With
HTML5
CSS3
JavaScript (Vanilla)
SVG Charts (no external libraries needed)

Concepts Demonstrated
Cloud infrastructure monitoring
CPU, memory and network metrics
Alert thresholds and status indicators
Multi-region cloud deployment awareness
Real-time data visualization
